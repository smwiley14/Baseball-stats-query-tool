# top-level packages). Partially pinning only the "main" packages let
# transitive deps (langgraph-prebuilt, langgraph-checkpoint, etc.) drift to
# newer, incompatible releases on a fresh `pip install` — that's what broke
# the last two production builds (ImportError: ExecutionInfo missing from
# langgraph.runtime, because a newer langgraph-prebuilt shipped expecting a
# newer langgraph core than the one pinned here). Full freeze avoids that
# class of bug entirely: what you build is byte-for-byte what was tested.
#
# To refresh deliberately: pip install --upgrade <package>, retest the app
# end-to-end, then `pip freeze` again and replace this file wholesale.
